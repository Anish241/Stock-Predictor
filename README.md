# Stock-Predictor

## How to Run ?

### Installations:
  #### cd into DjangoServer:
      pip install -r requirements.txt
  #### cd into website/client
      npm install
  #### cd into website/ server
      npm install
### Start:
  #### cd into DjnagoServer:
      python manage.py runserver
  #### cd into website/client
      npm run dev
  #### cd into website/ server
      npm start
